PRE-STABLE


Finally, the PRE-ESTABLE version of ANDRAX-NG has arrived.

Test what will be the last version before the long-awaited STABLE!

We are bringing many improvements thanks to the feedback given by the community, improvements that leverage the performance, number of tools and usability of the platform.

General changes:

  • HARD-BASED upgrade to 5.0
  • Python 3.13.3 accessible for tools
  • Massive upgrade of tools and utilities
  • AFOS-NG updated to version 1.0.1
  • Up-to-date and applied security patches
  • Updated Programming languages

Anti-Python Movement

SNAKE Security is joining the Anti-Python Movement promoted by companies such as Google to deprecate the use of the Python language as much as possible. It is well known that today Python has become a refuge for people who do not know how to program, which makes almost any system created in Python both inefficient and banal… for this reason ANDRAX-NG will only include Python-based tools that are exceptionally good and indispensable! In addition, we will give priority to tools developed in robust languages such as C, C++, GO, Rust, Ruby and so on.

Several Python tools that were not very useful or that had alternatives have been removed and/or replaced by their respective alternatives!

Android version

  • Updated installation method ( See the documentation for the new installation instructions )
  • Some interface improvements
  • “Permission denied” bug when opening Dragon Terminal on some manufacturers has been fixed ( Thanks to “The Matrix has you” and “Matteo” for helping! )
  • KernelSU compatibility problem solved ( Thanks to “ratnikov” for helping! )
  • Uninstall option added

Real Hacker's Keyboard

The very old and outdated “Hacker’s Keyboard” was a legendary project for the first versions of Android, but since Android 4.1 it has been outdated and long abandoned by its creator due to a lack of financial incentive from users.

However, we at SNAKE Security refuse to let good projects die - after all, something that benefits the community is always a good idea, and ideas are BulletProof!

We’re excited to announce “Real Hacker’s Keyboard” an updated fork for full compatibility with Android 15 that will be maintained by SNAKE Security, this new keyboard has many optimizations, clean code and a renewed interface.

Even though it’s a modernization, “Real Hacker’s Keyboard” is still compatible with older versions of Android!

The “Real Hacker’s Keyboard” APK file is included in the ANDRAX-NG Portable [.zip] file, enjoy!
