Welcome to ANDRAX-NG

At last the wait is over, the new generation of ANDRAX is here!

Since ANDRAX was first proposed in mid-2016 (ANDRAX-NG History), a lot has happened, some good, some bad...

This is the fourth generation of ANDRAX, and as you will see, this generation is ready for the new CyberSecurity testing era.

What's new?

First, the platform has been completely redesigned from the ground up to be compatible with new methodologies and/or techniques and to provide a better user experience for the community.

There are more changes than we can list (some changelog data has been lost over the years)

The Portable and Android versions have been merged into one package. The differences are the interface and the installation method!

Android/Portable:

  • A huge facelift
  • Huge improvements to the CodeHACK-IDE
  • Android 12+ Fixed (Shell stuck bug)
  • Android 14+ Fixed (Dragon Terminal Crash)
  • Large screen support (more than 12 inches) [Android]
  • Custom EKS add-on capability
  • Ability to compile Linux, Windows and MacOS x86_64 code directly from arm processors!
  • D.C.O Redesigned and Improved!
  • DBUS Fixed [Portable]
  • VNC Interface improved [Android]
  • New installer [Portable]
  • Dragon Terminal Improved [Android]

Desktop:

  • DBUS Fixed
  • Improved launcher
  • New installer

Common:

  • AFOS 100% operational
  • All tools upgraded
  • New DUAL-Update System
