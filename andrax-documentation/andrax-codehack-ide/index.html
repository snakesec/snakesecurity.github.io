CodeHACK IDE

The world's simplest IDE for Android

SNAKE Security TEAM

A simple IDE

Don’t expect anything special from CodeHACK, it’s probably the most basic IDE in the world…

CodeHACK is a simple integrated development environment that has been created for ANDRAX-NG, CodeHACK brings the most basic features to Android, allowing it to be a kind of “workaround” for those who need to code or script quickly and easily.

Even though CodeHACK is a very simple IDE, it has some interesting features like:

  • Supports 20+ programming languages
  • Bright and dark themes
  • Key binding for the most important IDEs and text editors
  • Auto-complete
  • Snippets
